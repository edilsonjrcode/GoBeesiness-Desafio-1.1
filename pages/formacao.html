Graduação Centro Universitário UNIESP

Sistemas para Internet

Graduação que visa formar profissionais capacitados para atuar no desenvolvimento de soluções para a web, desde websites estáticos até aplicações web complexas e sistemas de informação. Durante o curso, os estudantes são introduzidos aos fundamentos da computação e da internet, aprendendo sobre linguagens de marcação (como HTML) e estilização (como CSS) para criar interfaces web, além de linguagens de programação (como JavaScript) para adicionar interatividade e dinamismo aos sistemas. Além disso são ensinados conceitos de Orientação a Objetos, Desenvolvimento Backend com Java, Qualidade de Software utilizando Cypress para Testes Automatizados, DevOps
