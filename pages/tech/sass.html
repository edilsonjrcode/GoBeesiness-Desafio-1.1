Sass

Extensão do CSS

Uma ferramenta facilitadora

Sass é uma extensão do CSS3, adicionando regras aninhadas, variáveis, mixins, herança de seletor e muito mais. Ele é traduzido para CSS padrão e bem formatado usando a ferramenta de linha de comando ou um plug-in de estrutura da web. Sass é uma ferramenta na categoria Pré-processadores/Extensões CSS de uma pilha de tecnologia. Sass é uma ferramenta de código aberto com 14,9 mil estrelas do GitHub e 2,1 mil garfos do GitHub.
