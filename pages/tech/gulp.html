GulpJs

Biblioteca do Javascript

Automação com JS

Construa tarefas de automação do sistema: minificação e cópia de todos os arquivos JavaScript, imagens estáticas. Mais capaz de observar arquivos para executar novamente a tarefa automaticamente quando um arquivo é alterado. gulp é uma ferramenta na categoria JS Build Tools / JS Task Runners de uma pilha de tecnologia. gulp é uma ferramenta de código aberto com 32,9 mil estrelas do GitHub e 4,2 mil garfos do GitHub.

Site oficial: gulpjs.com
