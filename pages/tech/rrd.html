React Router

Rotas no React

Solução de roteamento no React

React Router é uma solução de roteamento completa projetada especificamente para React.js. Ele sincroniza facilmente os componentes do seu aplicativo com a URL, com suporte de primeira classe para aninhamento, transições e renderização no servidor. React Router é uma ferramenta na categoria JavaScript Framework Components de uma pilha de tecnologia. React Router é uma ferramenta de código aberto com Estrelas do GitHub eBifurcações do GitHub.

Site oficial: reactrouter.com/en/main/start/overview
