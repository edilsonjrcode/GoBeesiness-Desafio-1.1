HTML5

Linguagem de marcação

Tecnologia central da internet

HTML5 é uma linguagem de marcação de tecnologia central da Internet usada para estruturar e apresentar conteúdo para a World Wide Web. Em outubro de 2014, esta é a quinta revisão final e completa do padrão HTML do World Wide Web Consortium (W3C). A versão anterior, HTML 4, foi padronizada em 1997. HTML5 é uma ferramenta na categoria Idiomas de uma pilha de tecnologia.

Site oficial: developer.mozilla.org/pt-BR/docs/Web/HTML
